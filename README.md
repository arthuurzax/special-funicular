# WebEssence - Portfólio

## 📌 Sobre o Projeto
WebEssence é um site de portfólio moderno e responsivo, projetado para freelancers e empresas que desejam exibir seus serviços de forma profissional. O site inclui um modo escuro, seções organizadas e um design otimizado para melhor experiência do usuário.

## 🚀 Funcionalidades
- 💡 **Modo Claro/Escuro** para melhor experiência visual.
- 📜 **Seções:** Sobre, Portfólio, Contato.
- 📱 **Design Responsivo**, adaptado para todos os dispositivos.
- ⚡ **Código otimizado e estruturado.**

## 📂 Estrutura do Projeto
```
📂 WebEssence
│── 📁 public
│   └── 📁 images (Imagens do site)
│── 📁 styles
│   ├── global.css (Estilos globais)
│   ├── home.css (Estilos da página inicial)
│── 📁 scripts
│   ├── main.js (Modo escuro e interações)
│── index.html (Página principal)
│── README.md (Este arquivo)
```

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## 📥 Como Usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/WebEssence.git
   ```
2. Acesse a pasta do projeto:
   ```bash

   ```
